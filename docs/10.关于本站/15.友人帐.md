## 友链申请

欢迎和大家交换友链~ 🎉 

### 申请要求

✅ 贵站在中国大陆区域能够正常访问

✅ 贵站为个人博客类网站

✅ 贵站符合中国大陆法律法规

✅ 贵站已添加本站友情链接


友链格式和本站信息如下：

```YAML
- link: https://www.peterjxl.com
  avatar: https://image.peterjxl.com/blog/re0.jpg
  name: PeterJXL
  desc: 那就从零开始
  bgColor: #CBEAFA
```


### 申请方式

1. 提交评论
2. [点我发送邮件✉](mailto:peterjxl@qq.com)
3. 通过 GitHub 或 Gitee 的 PR （本文底部有个“在 GitHub 上编辑此页”的链接）
