# 00.Tomcat介绍

　　Tomcat 是常见的免费的 Web 服务器.  Tomcat 这个名字的来历，Tomcat 是一种野外的猫科动物，不依赖人类，独立生活。 Tomcat 的作者，取这个名字的初衷是希望，这一款服务器可以自力更生，自给自足，像 Tomcat 这样一种野生动物一般，不依赖其他插件，而可以独立达到提供 Web 服务的效果。
