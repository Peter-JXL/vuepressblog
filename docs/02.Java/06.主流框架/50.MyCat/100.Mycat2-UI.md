# 100.Mycat2-UI

　　Mycat2 UI 是官方推出的 Mycat2 监控工具。
<!-- more -->
　　‍

## 下载

　　[http://dl.mycat.org.cn/2.0/ui/](http://dl.mycat.org.cn/2.0/ui/)，选最新版下载即可

　　运行环境：在安装 JDK8 的环境，双击 jar 包就可以打开（或在命令行运行 java -jar xxx.jar）

　　‍

## 使用

　　（1）连接 Mycat2（需要 Mycat2 服务器正常启动）。
