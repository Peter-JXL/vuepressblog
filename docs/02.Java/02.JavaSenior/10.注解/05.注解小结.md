# 05.注解小结

![](https://image.peterjxl.com/blog/41-20221230194431-5krpgt1.jpg)


　　注解不是程序的一部分，可以理解为注解就是一个标签

　　注解给谁用？编译器，解析程序用

<!-- more -->　　‍

　　注解分类：

* JDK内置注解，例如@Override检验方法重写，@Deprecated标识方法过期等
* 自定义注解
* 第三方框架定义的注解比如SpringMVC的@Controller等

　　以后大多数时候，我们会使用注解，而不是自定义注解
