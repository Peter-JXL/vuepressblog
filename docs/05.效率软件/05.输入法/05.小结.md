# 05.输入法小结


　　输入法不在多，而在精，基本上所有输入法的功能都大同小异，弄懂一个后另一个也很容易上手。

　　输入法要深入去折腾也是很费时间的，并且大部分人不需要太过折腾，看了本系列的博客基本上够日常使用了。

　　后续视情况可能会将我学习五笔的笔记分享出来，或者再去学习一下双拼并分享。
