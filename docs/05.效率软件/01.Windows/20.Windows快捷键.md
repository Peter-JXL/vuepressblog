## 常用快捷键

　　接下来讲讲常用的快捷键（个人认为）。

　　‍

　　‍

### 基本操作

　　Ctrl + C（或 Ctrl + Insert）： 复制

　　Ctrl + V（或 Shift + Insert）： 粘贴

　　Ctrl + X： 剪切

　　Ctrl + Z： 撤消

　　Ctrl + Y： 恢复

　　Ctrl + A： 全选

　　Alt + Tab： 在打开的应用之间切换

　　Alt + F4： 关闭应用

　　F2： 重命名

　　F5： 刷新

　　F11：全屏

　　Delete（或 Ctrl + D）： 删除选定项，将其移至回收站

　　Shift + Delete： 删除选定项，无需先移动到回收站。

　　Ctrl + 向右键： 将光标移动到下一个字词的起始处。

　　Ctrl + 向左键： 将光标移动到上一个字词的起始处。

　　Ctrl + 向下键： 将光标移动到下一段落的起始处。

　　Ctrl + 向上键： 将光标移动到上一段落的起始处。

　　Ctrl + Shift + Esc： 打开任务管理器。

　　Shift 与任何箭头键： 在文件夹（或桌面）中选择多个项目，或在文档中选择文本。

　　Shift + Home： 选择光标处到该行开始的文本

　　Shift + End： 选择光标处到该行末尾的文本

　　‍

　　‍

### Win键相关

　　Win： 打开或关闭“开始”菜单。

　　Win + A 打开操作中心。

　　Win + B： 将焦点设置到通知区域

　　Win + L：  锁定电脑

　　Win + D： 显示和隐藏桌面

　　Win + 句号 (.) 或分号 (;)： 打开emoji输入面板

　　Win + 逗号：预览桌面

　　Win + C：打开小娜 Cortana

　　Win + E：打开文件资源管理器。

　　Win + I：打开Windows设置

　　Win + P：设置屏幕显示模式（多屏时）

　　Win + R：打开“运行”对话框。

　　Win + T：循环浏览任务栏上的应用

　　Win + V：打开剪贴板

　　Win + Pause：显示“系统属性”对话框

　　Win + 数字：打开任务栏中的第x个应用（x为数字）

　　Win + Shift + 数字：重复打开任务栏中的应用

　　Win + Tab：打开任务视图

　　Win + 向上键：最大化窗口

　　Win + 向下键：最小化窗口

　　Win + 向左键：向左分屏

　　Win + 向右键：向右分屏

　　Win + Shift + 向左键或向右键：将应用移动到另一台显示器

　　‍

### 文件资源管理器键盘快捷方式

　　Ctrl + N：打开新窗口

　　Ctrl + W：关闭活动窗口

　　Ctrl + 鼠标滚轮：放大/缩小图标

　　Ctrl + Shift + N：新建文件夹

　　Alt + P：显示预览面板。

　　Backspace：返回上级目录

　　‍

　　‍

　　‍

### 任务栏键盘快捷方式

　　‍

　　Shift + 单击任务栏按钮：打开应用或快速打开另一个应用实例。

　　Ctrl + Shift + 单击任务栏按钮：以管理员身份打开应用。

　　Ctrl + 单击分组任务栏按钮：循环浏览组的窗口（例如切换多个记事本），示意gif：
