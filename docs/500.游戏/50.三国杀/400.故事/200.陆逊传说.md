# 陆逊传说

壮哉神将陆伯言，丈八二桃把营联。

<!-- more -->

本方残血放万箭，敌人势弱开桃园。

一兵队友张文远，二兵袁术只等闲。

转眼队友全托管，挂上闪电继续连。
